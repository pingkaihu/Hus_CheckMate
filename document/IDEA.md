# IDEAS.md

## 核心特色

- 結合 PPG 與 Checklist
- 獨特角色美術設計

## 基本功能

### 角色數值系統
- 不要有太複雜的角色屬性
- 一個角色可以開啟多個職業
- 每個職業有自己獨立的等級、經驗值、任務、技能
  有其各自不同的階段
- 出版預計製作兩個一般職業、一個隱藏職業
- 一般職業: 戰士、學者
- 隱藏職業: 造物者
- 戰士路線: 新手 -> 見習戰士 -> 戰士 -> 菁英戰士 -> 戰鬥大師
- 學者路線: 新生 -> 學院生 -> 學者 -> 研究者 -> 賢者

### 任務系統 (Checklist System) 
- 完成任務會提供經驗值
- 任務類型: 每日任務、每週任務、自訂任務
- 每日任務: 每天中午12點重置，12點過後就不能修改當日選擇的任務-
- 每日任務範例: '戰士' 重訓、有氧 '學者' 閱讀、寫作
- 每週任務: 每週一中午12點重置，12點過後就不能修改當週選擇的任務
- 每週任務範例: 聚會(或許可以考慮需要朋友協助完成)、聽演講、看展
- 自訂任務: 可以自訂任務，但有上限

- 根據角色等級、職業，提供不同可添加的任務，例如戰士可添加重訓、有氧(能量系統訓練)
等，學者可添加閱讀、寫作等任務
- 真實情境: 任務對應到真實的人類情境，例如戰士最初的任務可以是重訓20min，隨著等級提升，現實中也養成運動習慣，Lv5之後可能任務變為重訓30min，也提供更多經驗值
- 任務變化: 以戰士任務為例，重訓後需要充足休息，連續重訓三天後會出現睡滿8小時的任務可以添加


### 紀錄與成就系統
- 任務回顧: 提供任務記錄回顧、統計數據，
- 參考 Duolingo 的成就系統
- 多個職業並進有機會達成"文藝復興"
